# 📊 Dashboard de Clientes y Ventas

## 🎯 Objetivo

Este proyecto tiene como objetivo analizar el comportamiento de clientes y ventas mediante un dashboard interactivo en Excel.

---

## 📁 Datos utilizados

El conjunto de datos incluye:

* Identificador de usuario
* Tipo de suscripción
* Ingresos mensuales
* Fecha de suscripción

Se han generado variables adicionales como:

* Mes de suscripción
* Año de suscripción

---

## ⚙️ Proceso

1. Limpieza y preparación de datos
2. Creación de nuevas variables
3. Construcción de tablas dinámicas
4. Desarrollo del dashboard interactivo

---

## 📈 Análisis realizado

El dashboard permite analizar:

* Ventas por mes
* Evolución de ingresos en el tiempo
* Distribución por tipo de suscripción
* Top clientes
* Clientes con mayor aportación de ingresos

---

## 🎛️ Interactividad

El dashboard incluye filtros dinámicos por:

* Año de suscripción
* Mes de suscripción
* Tipo de suscripción

---

## 📊 Conclusiones

* Existen meses con mayor volumen de ventas
* Hay diferencias entre tipos de suscripción
* Un grupo reducido de clientes concentra gran parte de los ingresos

---

## 🛠️ Herramientas utilizadas

* Excel (tablas dinámicas, gráficos y segmentación de datos)
* Python (procesamiento de datos)
